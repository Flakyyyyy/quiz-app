## 📖 О проекте

**QuizMaster Pro** — это одностраничное веб-приложение (SPA) для проведения викторин.  
Пользователи могут:

- 🎯 Проходить готовые тесты с ограничением по времени
- 📊 Получать детальную статистику и разбор ответов
- ✏️ Создавать, редактировать и удалять собственные тесты
- 📥 Импортировать и экспортировать тесты в формате JSON
- 💾 Хранить данные в локальном хранилище браузера (LocalStorage)

Приложение имеет современный интерфейс в тёмной цветовой гамме с эффектами стекла, градиентами и плавными анимациями.

---

## 🚀 Функциональные возможности

### 🧩 Для пользователя (игрока)
- Выбор теста из списка доступных
- Отображение прогресса и оставшегося времени
- Подсчёт очков, правильных/неправильных ответов и серии верных ответов
- Мгновенная обратная связь после каждого ответа
- Детальная страница результатов с процентом, временем, максимальной серией и разбором каждого вопроса

### 🛠️ Для создателя тестов
- Создание нового теста с настраиваемым количеством вопросов и вариантов ответа
- Установка лимита времени на каждый вопрос
- Редактирование и удаление существующих тестов
- Импорт теста из JSON‑файла (например, скачанного или созданного вручную)
- Экспорт созданного теста в JSON для обмена или резервного копирования

### 💾 Работа с данными
- Все тесты сохраняются в `localStorage` браузера
- При первом запуске загружаются предустановленные тесты из файла `initial-quizzes.json`
- Для каждого теста хранится лучший результат (в процентах), который отображается в списке

## 🛠️ Технологии

- **HTML5** — семантическая вёрстка
- **CSS3** — кастомные свойства, flex/grid, анимации, адаптивность
- **JavaScript (ES6+)** — модульная архитектура
- **LocalStorage API** — клиентское хранение данных
- **FileReader API / Blob** — импорт и экспорт JSON‑файлов
- **Git & GitHub** — контроль версий и хостинг

## ⚙️ Установка и запуск

### 1. Локальный запуск (рекомендуется для разработки)

Из-за использования ES6‑модулей приложение требует запуска через HTTP‑сервер.  
Самый простой способ — использовать **Live Server** в VS Code:

1. Установите расширение **Live Server** (Ritwick Dey).
2. Откройте папку проекта в VS Code.
3. Нажмите правой кнопкой на `index.html` → **Open with Live Server**.
